2022/8/20

网页的换肤效果终于写完了（
用的是<link rel="stylesheet alternate" href="***.css" title="***">解决。其中：
 rel只有stylesheet且无title的，必然加载并渲染；
 rel只有stylesheet且title不为空的，默认加载并渲染；
 rel有alternate、stylesheet且title不为空的，默认加载但不渲染；
可以用下面的JS代码使不渲染的渲染： 

 🔴 🟠 🟢 
document.querySelector('link[href="***.css"]').disabled=false;

踩了个大坑：用js使不渲染的渲染，实际上disabled属性需要先设为true，再设为false才能使其渲染！

还有一个新知识点：原生js实现块内页面横向滚动：

 🔴 🟠 🟢 
let codes = document.getElementsByTagName("pre");
for (let i = 0; i < codes.length; i++) {
const element = codes[i];
element.addEventListener("wheel", (event) => {
    event.preventDefault();
    element.scrollLeft += event.deltaY;
});
}

    loginwind.style.top = ev.clientY - y + 'px';

        document.onmousemove = null;
